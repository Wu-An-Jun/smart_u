## AI助手返回格式

当用户表达跳转意图时，AI助手应返回格式：
```json
{
  "action": "navigate",
  "page_code": "编号",
  "page_name": "页面名称",
  "reason": "跳转理由"
}
```

## 特殊情况处理

1. **模糊表达**: 当用户表达不明确时，AI应询问具体需求
2. **多重匹配**: 当一个表达可能对应多个页面时，优先选择最相关的
3. **无匹配**: 当无法匹配到具体页面时，返回首页(编号: 10)或提供相关建议
